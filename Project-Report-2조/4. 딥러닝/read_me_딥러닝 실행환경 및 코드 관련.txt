딥러닝 실행환경 : 구글 Colab (Colaboratory)

엔디비2조딥러닝코드.ipynb를 열고 아래 csv와 xlsx를 해당 경로 혹은 colab 상에 넣은 후 코드를 실행합니다.
(셀이 두가지로 나누어져 있습니다.
첫번째 셀 : 딥러닝 학습 및 검증 - 딥러닝사용파일.csv 사용
두번째 셀 : 모델 테스트 - 검증.xlsx 사용)

딥러닝 학습에 사용된 input csv 파일 : 딥러닝사용파일.csv
딥러닝 모델 테스트에 사용된 input xlsx 파일 : 검증.xlsx


결과 데이터는 따로 파일로 export되지 않고 위 코드 상에서 print 하여 나타내어집니다. 
(여러 그래프들과 결과값 과정이 프린트되어지기 때문입니다. )
프린트 된 결과창을 output 폴더에 txt파일과 png 파일로 첨부하였습니다.
